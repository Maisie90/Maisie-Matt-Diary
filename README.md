# Diary App
Maisie and Matthew have built their own diary for a hackathon. It lets users create, view, edit, and delete diary entries. Using a SQL database, along with HTML,CSS, and JS.

## Bugs

Lots of bugs still, front end not connected to back end. Backend working ok. .env file not ignored so you can connect to database as no time left to add instructions. 

## Features
- Add a diary entry with category and text
- View all entries in order of recency
- Edit entry text
- Delete entries
## User stories
As a user, I should be able to:
-Create a diary entry with a specific date and time, text and category (DONE)
-See a list of my diary entries and sort them by recency
-See the details of a specific diary entry
-Search for a specific diary entry by date/month/year or category
-Update the text of a diary entry (DONE)
-Delete a diary entry (DONE)
## Tech needed // used
Node.js, Express, SQL, HTML, CSS and JavaScript.
## Running the project
Install dependencies:
npm install
node db/setup.js
npm strat
Open index.html in live server
